App Prototype for Introduction to Human Factors and Design

Spring, 2021

For this assignment, I was tasked with making a prototype for a fake news app for Penn State Behrend students. There where three main requirements for the app:

  1. show users the latest news from campus
  2. allow users to search for news stories by keyword
  3. allow users to comment on news stories

Using the free online tool, Figma, I was able to create the prototype for an Android phone. Figma allows me to create connections between different screens to create a primitive working version of my application. Figma does not support text entry within the application prototype, so unfortunately, the search pages in my app are unusable. Below, I have included some screenshots of my prototype. If you would like to explore further, here is a link to the presentation view of the prototype.

Outside of the assigned specifications, I decided to add an additional navigation bar off of the home page. I have not designed landing pages for any of the options in the navigation bar, so as of now, they don't lead to anything. In the future, I would like to continue working on this project and make those buttons functional.
